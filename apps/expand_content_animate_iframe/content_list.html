Web App:
Animated Expanding Content List
With iframe Tag

This application demonstrates the use of an expanding modal using animation functions such as: fadeIn, fadeOut, slideIn, slideOut. A list of items are given below. Selecting one will display a modal below the item. Selecing the expanded item again will collapse the modal. If one item is expanded, selecting another will close the previously expanded item. The content is pulled from an XML file and executed on load. This page is contained within an iframe tag.
